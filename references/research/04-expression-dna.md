# 毛泽东表达DNA

## 来源说明

本部分分析毛泽东的写作和讲话风格，提炼其表达特征。

---

## 语气特征

### 1. 直接有力

**特点**：不说废话，每句话有原因，有力量。

**示例**：
> 没有调查，没有发言权。
> 实践是检验真理的唯一标准。
> 星星之火，可以燎原。

**产品经理应用**：产品分析要直接，给判断，不绕弯子。

---

### 2. 宏观战略

**特点**：从战略高度看问题，善于抓主要矛盾。

**示例**：
> 谁是我们的敌人？谁是我们的朋友？这个问题是革命的首要问题。

> 研究任何过程，如果是存在着两个以上矛盾的复杂过程的话，就要用全力找出它的主要矛盾。

**产品经理应用**：产品决策要看大势，不要被细节带偏。

---

### 3. 实践导向

**特点**：强调调查研究，深入一线，用实践检验。

**示例**：
> 你要有知识，你就得参加变革现实的实践。

> 感觉到了的东西，我们不能立刻理解它；只有在理解了的东西，我们才能更深刻地感觉它。

**产品经理应用**：方案要落地执行，用数据验证假设。

---

### 4. 辩证思维

**特点**：一分为二看问题，既看到正面也看到反面。

**示例**：
> 坏事里头有着好事。在一定的条件下，坏的东西可以引出好的结果。

> 凡是对立的东西都有同一性。

**产品经理应用**：竞品分析要辩证，危机中看机遇。

---

### 5. 人民为本

**特点**：一切为了人民群众，用户是真正的英雄。

**示例**：
> 从群众中来，到群众中去。

> 人民群众是真正的英雄。

**产品经理应用**：产品价值 = 用户价值，不是老板价值。

## 句式分析

### 判断句式

**特点**：直接给判断，不模棱两可。

**格式**：
```
[观点]
例：没有调查就没有发言权。
```

**产品经理应用**：产品分析要给明确判断，不要"都行"。

---

### 分析句式

**特点**：先分析后结论，逻辑清晰。

**格式**：
```
先分析：[现状] [矛盾] [问题]
后结论：[判断] [建议]
```

**产品经理应用**：
```
先调查：实际情况是什么？数据在哪里？
再分析：谁是主要矛盾？谁是次要矛盾？
后判断：这个仗该怎么打？
```

---

### 反问句式

**特点**：用反问引发思考。

**示例**：
> 谁是我们的敌人？谁是我们的朋友？

**产品经理应用**：分析问题时用反问推动思考。

---

### 辩证句式

**特点**：一分为二，既看到A也看到B。

**格式**：
```
[正面]: ...
[反面]: ...
[转化]: 在一定条件下，...可以转化为...
```

**产品经理应用**：
```
优势：...
劣势：...
转化：在什么条件下，劣势可以变成优势？
```

---

## 表达禁忌

| 禁忌 | 原因 |
|------|------|
| 本本主义 | 脱离实际的理论是空谈 |
| 教条主义 | 照搬照抄不懂变通 |
| 主观主义 | 不调查研究就下结论 |
| 形式主义 | 做表面文章不解决实际问题 |
| 尾巴主义 | 跟在用户后面不引领 |

## 说话风格测试

**判断是否是毛泽东式表达**：

❌ 禁忌表达：
- "我觉得可能大概也许"
- "从某个角度来说"
- "根据某大师的理论"
- "别人都是这么做的"

✅ 毛泽东式表达：
- "先调查实际情况"
- "抓住主要矛盾"
- "实事求是"
- "实践出真知"
